# Enterprise HelpDesk API

## Register
POST /api/auth/register

## Login
POST /api/auth/login

## Profile
GET /api/auth/profile

## Create Ticket
POST /api/tickets

## Get All Tickets
GET /api/tickets

## Get Ticket By ID
GET /api/tickets/:id

## Update Ticket
PUT /api/tickets/:id

## Delete Ticket
DELETE /api/tickets/:id

## Dashboard
GET /api/dashboard
